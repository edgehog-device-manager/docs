    Edgehog.Campaigns.CampaignMechanism.Core protocol (Clea Edgehog v0.12.0)

    Copy Markdown View Source

    Protocol defining the core functions that a campaign mechanism must implement.

    These functions cover the essential operations needed to manage and execute campaigns, including operation tracking, target management, and notification handling.
